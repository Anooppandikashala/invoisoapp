Data, Backup & Import

Export, import, and protect your billing data.

Backup and Restore

Invoiso stores all data in a local SQLite database. Use the built-in backup to export a copy and restore it on any machine.

CSV Import & Export

Import customers and products from CSV files. Export your catalogue for offline editing or migration.

Offline Data Privacy

All data stays on your device. Invoiso never connects to external servers. Your billing data is entirely private.
Platform Setup

Installation and compatibility notes per OS.

Linux Installation

Use the AppImage (no install needed) or the .deb package. Tested on Ubuntu 22.04, 24.04 LTS, Debian, and Fedora.

Windows Installation

Download the .exe installer. Run it and follow the setup wizard. No admin account needed in most configurations.

macOS Installation

Download the .dmg file, open it, and drag Invoiso to Applications. If Gatekeeper blocks it, go to System Settings → Privacy & Security → Open Anyway.

Quick Answers

Is Invoiso free to use?

Yes. Invoiso is 100% free with no subscription, no usage limits, and no paid tier. See the pricing page for details.

Where is my data stored?

All data — customers, invoices, products, settings — is stored in a SQLite database file on your local device. Nothing is sent to any server.

Does it work without internet?

Completely. Invoiso is designed for full offline use. No internet connection is ever required.

Can I move my data to another computer?

Yes. Use the built-in backup function to export your database, then restore it on the new machine after installing Invoiso.

Where do I report a bug or request a feature?

Open an issue on GitHub. All development happens in the open.
